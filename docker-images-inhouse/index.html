    In-house Docker images

    In order to access these images from our in-house registry, you need to log in using the following credentials:

    docker login -u public -p public public.aml-repo.cms.waikato.ac.nz:443

    When pulling images, you need to ensure that you are using the following prefix (the pull commands do not show this, due to using a reverse proxy):

    public.aml-repo.cms.waikato.ac.nz:443/

    For some of the frameworks, we have tutorials available on how to use them in practice. This includes data preparation, training and making predictions.

    https://www.data-mining.co.nz/applied-deeplearning/
