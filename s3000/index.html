    Our commercial S3000 software is aimed at environmental laboratories that analyze samples such as soil, plant, feed and manure using spectroscopy (NIR, MIR, LIBS or XRF).

    Instead of using calibrations that the manufacturers sell with their instruments, our system offers a bigger range of pre-processing techniques and sophisticated algorithms (due to our background in machine learning).

    Any calibrations that you build using your own data will continue being your intellectual property (IP). With the system operating on-premise, your data will stay safe as well.

    The software is based on our ADAMS workflow software and offers the following benefits:

    Use Cases

    • Calibrations built from in-house data
    • Integration into business processes, e.g., in-house LIMS
    • Data fusion support, e.g., NIR+XRF
    • Instrument conversions, between models or manufacturers

    Diagrams

    Operation

    • Unlimited number of calibrations (Weka/Python)
    • Unlimited number of import tasks
    • No additional cost in making predictions
    • Automated/scheduled training
    • Provenance for tracing samples through the system
    • Flag unreliable predictions
    • Generate PDF report from settings and performance of calibrations
    • Export of parameter settings for accreditation purposes

    Support

    • Expert assistance for set up
    • Tailored contract for ongoing support
    • Training of staff
    • Documentation of customizations
    • Calibration optimizations
    • Stability and reliability of provider

    Contact us

    Platform

    • Open-source, cross-platform code base written in Java
    • Extensible plugin-architecture for client-specific customizations
    • Ability to add custom tasks via external ADAMS workflows
    • Scripting support via Groovy
    • Java frontend: configuration of calibrations and tasks
    • Web frontend: administrative and monitoring tasks
    • 24/7 operation of prediction backend

    Data

    • Incorporate wet chemistry co-variables to improve calibrations
    • Supports common spectral file formats like CAL, JCamp-DX, NIR, Opus, SPA, SPC, spreadsheet-based, Unscrambler
    • Supports data retrieval from databases like MS SQL Server, MySQL, PostgreSQL via JDBC
