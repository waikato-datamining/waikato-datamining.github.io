Docker images for `Yolo26 <https://github.com/ultralytics/ultralytics/>`__ are now available.

The code used by the docker images is available from here:

`github.com/waikato-datamining/pytorch/tree/master/yolo26 <https://github.com/waikato-datamining/pytorch/tree/master/yolo26>`__

The tags for the images are as follows:

* In-house registry:

  * ``harbor.cms.waikato.ac.nz/public/pytorch/pytorch-yolo26:8.4.16_cuda12.6``
  * ``harbor.cms.waikato.ac.nz/public/pytorch/pytorch-yolo26:8.4.16_cpu``

* Docker hub:

  * ``waikatodatamining/pytorch-yolo26:8.4.16_cuda12.6``
  * ``waikatodatamining/pytorch-yolo26:8.4.16_cpu``

The tutorial on *object detection* is available from here:

`www.data-mining.co.nz/applied-deep-learning/object_detection/yolo26/ <https://www.data-mining.co.nz/applied-deep-learning/object_detection/yolo26/>`__
