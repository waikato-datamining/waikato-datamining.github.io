New Docker images are available for the `MMDetection <https://github.com/open-mmlab/mmdetection>`__ object detection
framework, using the 3.1.0 release of MMDetection (code base as of 2023-06-30):

* `CUDA 11.3 <https://github.com/waikato-datamining/mmdetection/tree/master/3.1.0_cuda11.3>`__
* `CPU <https://github.com/waikato-datamining/mmdetection/tree/master/3.1.0_cpu>`__ (inference only)
