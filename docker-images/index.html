    Docker images

    For our own projects, we make a lot of use of Docker, to better manage the varying (and often clashing) requirements for CUDA and Python libraries. We not only offer a quick introduction to docker for data scientists, but we also make (most of) our Docker images freely available (see below).

    Registry

    In order to access these images, you need to log in our in-house registry using public credentials:

    docker login -u public -p public public.aml-repo.cms.waikato.ac.nz:443
